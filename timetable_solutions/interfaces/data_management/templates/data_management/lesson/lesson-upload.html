    Required file structure
    lesson_id subject_name teacher_id pupil_ids classroom_id total_required_slots total_required_double_periods user_defined_slot_ids
    maths-y1-a Maths 1 1; 2; 3 10 5 1 6; 24
    • lesson_id*: A text identifier used to identify this lesson. Note that these must be unique
    • subject_name*: The name of the lesson subject
    • teacher_id*: The ID of the lesson's teacher
    • pupil_ids*: IDs of the pupils to add to the lesson. Separate each id with a semicolon as shown in the example table
    • classroom_id*: The ID of the lesson's classroom
    • total_required_slots*: The number of timetable slots this lesson should be assigned each week. Note that this count should include any that you assign yourself
    • total_required_double_periods*: The number of double periods this lesson should be assigned each week. Note that this count should include any double periods you assign yourself, and does count towards fulfilling total_required_slots. Therefore it must be at most half of total_required_slots
    • user_defined_slot_ids*: IDs of the slots to add to the lesson. Separate each id with a semicolon as shown in the example table
    • *required - no empty values are allowed in these columns
